# 🌏 Tradutor PT/EN/ES → Chinês Simplificado

Aplicativo Windows que traduz texto digitado para Chinês Simplificado ao pressionar **Shift + Enter**.

## 🚀 Funcionalidades

- ✅ Tradução global (qualquer campo de texto)
- ✅ Ícone dinâmico (verde/vermelho)
- ✅ Seleção de idioma (Português, Inglês, Espanhol)
- ✅ Salva configurações automaticamente
- ✅ Executável portátil (.exe)

## ⌨ Atalho

Shift + Enter → Traduz linha atual
